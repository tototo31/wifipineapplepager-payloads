# Title: Live Walk
# Author: MusicalVR
# Description: Actively maps and logs wifi networks using aircrack-ng. Handshake capture is disabled to save space.
# Category: Recon
# Dependencies: aircrack-ng (install via 'opkg update && opkg install aircrack-ng')
# Built for the Hak5 Wifi Pineapple Pager

YOU MUST HAVE AIRCRACK-NG  INSTALLED

Use:
opkg update
opkg install aircrack-ng
To install the desired tools needed. 

YOU MUST KNOW, This continuously runs until you top it, you will have the log of the last minute. 
It saves logs to live_walk  

This is a Live Recon tool for the Hak5 WiFi Pineapple Pager

I built this with the set it and forget it mentality while out and about. 

You can look at the logs live as they update every so often on your mobile device or just check back and see what you got at your home base!

As you walk it logs all the information found in the screenshot.
